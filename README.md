# Autism Risk Prediction using Machine Learning

This project focuses on predicting the risk of Autism Spectrum Disorder (ASD) using machine learning techniques based on questionnaire-based behavioral data.

## Overview
Autism Spectrum Disorder is a developmental condition that can be identified through behavioral patterns. This project applies machine learning models to analyze questionnaire responses and predict the likelihood of autism risk.

## Objectives
- Build predictive models for autism risk detection
- Compare performance of different machine learning algorithms
- Provide model explainability using SHAP

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- CatBoost
- LightGBM
- SHAP
- Matplotlib

## Machine Learning Models
- CatBoost
- LightGBM

## Features
- Data preprocessing and cleaning
- Feature analysis
- Model training and evaluation
- Model explainability using SHAP
- Performance evaluation using accuracy, precision, recall, and ROC-AUC

## Results
The models were evaluated to identify the most effective algorithm for predicting autism risk while maintaining interpretability.

## Future Improvements
- Hyperparameter optimization
